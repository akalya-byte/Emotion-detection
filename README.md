# Emotion Detection & Safety Monitoring System

## Overview
This project presents a smart system that integrates wearable sensors, machine learning, and mobile technology to detect human emotions in real time while ensuring user safety.

It analyzes physiological signals such as heart rate variability (HRV), galvanic skin response (GSR), and motion data to classify emotional states and trigger safety mechanisms during critical situations.

## Features
- Real-time emotion detection
- Physiological signal monitoring (HRV, GSR, Accelerometer)
- Machine learning-based classification
- SOS emergency alert system
- Caregiver access for real-time monitoring
- GPS tracking
- Emergency alarm activation
- Mobile app integration

## Technologies Used
- Machine Learning (Random Forest, SVM, LSTM)
- Data Processing & Analysis
- Wearable Sensor Data (HRV, GSR)
- Mobile Application Integration

## Problem Statement
Existing systems focus either on emotion detection or personal safety, but not both. This project combines both aspects into a single intelligent system for better user support.

## Solution
A hybrid system that monitors emotional states and triggers real-time safety alerts such as SOS notifications to caregivers, enabling quick response and continuous monitoring.

## Applications
- Mental health monitoring
- Personal safety systems
- Healthcare monitoring
- Emergency alert systems

## Future Enhancements
- Integration with IoT devices
- Improved deep learning models
- Offline functionality
- Enhanced privacy and security

## Conclusion
This project demonstrates how AI and wearable technology can be combined to improve both emotional well-being and personal safety through real-time monitoring and emergency response features.
